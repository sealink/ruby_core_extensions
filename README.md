# DESCRIPTION

Extends the core ruby classes with helpful functions

# INSTALLATION

Add to your Gemfile:
gem 'ruby_core_extensions'

# Extensions

The following classes are extend in this gem:

* Array
* BigDecimal
* Class
* Date
* DateTime
* Enumerable
* File
* Fixnum
* Hash
* Kernel
* Numeric
* Object
* Range
* String
* Time
